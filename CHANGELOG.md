# 변경 사항 (Changelog)

## v2.2.0 - 2025-12-25

### ✨ 인트로 화면 패럴랙스 효과 대폭 강화

#### 역동적인 이미지 움직임
- **움직임 범위 확대**: 50px → 120px (2.4배 증가)
- **부드러운 이징**: 선형 이동 → 0.08 이징으로 자연스러운 움직임
- **다양한 깊이**: 3단계 → 5단계 (0.3, 0.5, 0.7, 0.9, 1.0)
- **개별 속도**: 각 이미지마다 0.8~1.2 랜덤 속도

#### 3D 효과 추가
- **3D 회전**: rotateX, rotateY, rotateZ 적용
- **원근감**: perspective 1200px → 1500px
- **동적 스케일**: 마우스 거리에 따라 1.0~1.2배 크기 변화
- **뒷면 숨김**: backface-visibility로 깔끔한 회전

#### 시각적 개선
- **동적 투명도**: 깊이에 따라 0.6~1.0 변화
- **그림자 강화**: 20px → 25px, 호버 시 40px → 120px
- **이미지 필터**: brightness(1.05) + contrast(1.1)
- **GPU 가속**: will-change, translateZ(0) 적용

#### 성능 최적화
- requestAnimationFrame으로 60fps 유지
- transform 속성만 사용 (reflow 방지)
- backface-visibility로 렌더링 최적화

### 📦 파일 변경
- `app.js`: createIntroFloatingImages() 함수 완전 재작성
- `style.css`: 3D transform, perspective, 애니메이션 속성 추가

## v2.0.0 - 2025-12-25

### 🆕 새로운 기능

#### 1. 마우스 반응 이미지 효과
- 인트로 화면에 월별 대표 이미지들이 배치됨
- 마우스 움직임에 따라 이미지들이 패럴랙스 효과로 반응
- 각 이미지마다 다른 깊이(depth)로 다층 효과 구현

#### 2. 월별 대표 이미지 배치
- 각 슬라이드에 최대 3개의 이미지 배치 가능
- 이미지 위치가 슬라이드마다 자동으로 변경 (8가지 위치 패턴)
- 자연스러운 회전 효과와 그림자로 입체감 표현
- 호버 시 확대 효과

#### 3. 동영상 재생 기능
- MP4 동영상 자동 재생
- 루프 재생 및 음소거 설정
- 슬라이드 내 자연스러운 배치

#### 4. 클릭으로 슬라이드 이동
- 화면 아무 곳이나 클릭하면 다음 슬라이드로 이동
- 우측 하단에 힌트 메시지 표시
- 버튼 영역은 기존 기능 유지

#### 5. 떠다니는 배경 아이콘
- 월별 이모지 아이콘들이 배경에서 둥둥 떠다님
- 4가지 다른 애니메이션 패턴
- 랜덤한 크기, 위치, 속도

### ✨ 개선사항

- data.js에 `images`와 `video` 필드 추가
- 모든 월별 데이터에 샘플 이미지 URL 추가 (Unsplash)
- 5월 데이터에 샘플 동영상 추가
- 모바일 반응형 최적화 (이미지/동영상 크기 자동 조정)
- README.md에 이미지/동영상 추가 가이드 작성

### 🎨 스타일 변경

- 인트로 이미지 스타일링 (둥근 모서리, 그림자, 반투명 테두리)
- 슬라이드 미디어 위치별 스타일 (8가지 패턴)
- 호버 효과 추가
- 모바일 최적화 CSS

### 📦 파일 변경

- `index.html`: 인트로 이미지 컨테이너 추가
- `app.js`: 이미지 생성 및 마우스 반응 로직 추가
- `style.css`: 미디어 스타일링 및 반응형 CSS 추가
- `data.js`: 모든 월 데이터에 이미지/동영상 필드 추가
- `README.md`: 사용 가이드 업데이트
